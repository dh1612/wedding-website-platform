# Wedding Brochure Plan

Audience:
Wedding couples in Ireland looking for a tasteful, low-stress digital wedding experience.

Objective:
Position the product as a premium done-for-you service, not a generic website template or one-off AI mockup.

Narrative Arc:
1. Introduce the product as a modern wedding website and planning portal.
2. Show the blend of beauty, clarity, and convenience.
3. Present three clean package tiers with confident pricing.
4. Explain why this is different from generic templates or DIY AI output.
5. Show a simple onboarding process that feels easy and low-stress.
6. End with a clear invitation to enquire.

Slide List:
1. Cover: elegant service-led introduction
2. Product overview: what couples actually get
3. Packages: Essential Website, Website + AI Concierge, Complete Wedding Portal
4. Feature comparison: what is included in each tier
5. Why it is different: bespoke-feeling structure, guest management, planning tools
6. Process: simple 4-step delivery flow and enquiry prompt

Source Plan:
- Use the actual product capabilities built in this workspace as the primary source.
- Use the agreed package pricing from the conversation:
  - Essential Website: EUR195
  - Website + AI Concierge: EUR345
  - Complete Wedding Portal: EUR595

Visual System:
- Editorial, calm, refined brochure aesthetic
- Winter botanical and soft stone palette
- Deep evergreen, ivory, mist, muted gold accents
- Serif display with clean body typography
- Quiet negative space, layered cards, premium paper feel

Imagegen Plan:
- One text-free 16:9 art-direction plate per slide
- Slide 1 sets the visual language
- Slides 2-6 vary composition while keeping the same deck family

Asset Needs:
- No external logos needed
- No screenshots required for v1 brochure
- Use abstract wedding editorial visuals rather than literal UI screenshots

Editability Plan:
- All copy, pricing, package names, and comparison content authored as editable slide text
- Art plates used only as background direction
